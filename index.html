HTML TAGS

TAG NAME

ELEMENT

DESCRIPTION
PARAGRAPH tag <p> This tag is used for writing any kind of paragraph.
ANCHOR tag <a> ANCHOR tag creates a link to other internet location, or file.
ACRONYM tag <acronym> ACRONYM tags defines an acronym.
ADDRESS tag <address> ADDRESS tags are used to identify the author's contact information
BOLD tag <b> BOLD tag is specifying bold section within the text document.
BREAK tag <br> Line Break tag is specifying a new line
UNDERLINE tag <u> This tag is used for underline any text.
ITALIC tag <i> This tag is used for making any text ilatic.
HR tag <hr> HR tag draws a horizontal break line.
IMG tag <img> IMG tag attributes an image.
LINK tag <link> LINK tag defines a link to an external document, such as External Style Sheets.
LIST tag <li> LI tag defines a list of ordered and unordered items.
QUOTE tag <q> This tag is specifing short quotations.
STRIKE THROUGH tag <s> This tag produces a strike throughout a text.
TABLE tag <table> TABLE tag is defining a table.
TD tag <td> TD tag creates a data cell.
TR tag <tr> TR tag creates a row in a table.
TITLE tag <li> TITLE tag declares a title of an HTML document.
SUB tag <sub> SUB tag is defining a subscripted text.
